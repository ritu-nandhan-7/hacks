# Habo Space
---
*“Habits don’t form in 21 days. They form when your system makes failure impossible to ignore.”*

**Habo Space** is a group-first, socially aware habit-building platform that transforms habits from private checklists into shared, emotionally engaging experiences.

Unlike traditional habit trackers that rely on streaks and self-discipline, Haunt leverages behavioral psychology, social accountability, and gamification to create a system where habits are visible, consequential, and persistent.

## Key Features
- Low-friction habit tracking
- Socially visible progress
- Gamified but meaningful feedback
- Recovery-focused design
- Group accountability mechanisms
- Dynamic, non-linear habit growth

### Prerequisites
Make sure you have:
- Node.js (v18+ recommended)
- npm or yarn
- Git
- MongoDB (local or cloud like Atlas) OR Firebase setup

#### Clone the repository
```
git clone https://github.com/pavithra-ayeram/hacks.git
cd haunt
```

#### Project Structure
```
hacks/
├── server/            
└── README.md
```

#### Install Dependencies
```
cd server
npm install
```

#### Run the application
```npm run dev```

#### Test the server
- Open browser -> `http://localhost:5173`
- Create an account
- Explore the app
- Have fun tracking your habit!
